# 🛡️ TruthPulse AI – Fake News Detection Platform

A modern and responsive frontend prototype for an AI-powered Fake News Detection platform designed to combat misinformation through an intuitive and user-friendly interface.

🔗 https://fake-news-detection-orcin-zeta.vercel.app/

## 📖 About the Project

TruthPulse AI is a frontend prototype of a Fake News Detection platform that enables users to verify the authenticity of news articles through a clean and interactive interface.

The project focuses on creating an engaging user experience while laying the foundation for future integration with Artificial Intelligence and Machine Learning models capable of detecting fake news in real time.

Currently, the application demonstrates the complete user journey from onboarding to news verification and result visualization using a modern and responsive web interface.

## 🎯 Project Objective

The primary objective of TruthPulse AI is to provide users with a simple and accessible platform for identifying potentially fake or misleading news.

This project demonstrates the frontend architecture and user experience of a fake news detection system while preparing the application for future AI and backend integration.
